# 🌌 Solar System Planet Picker (Keyframe Animations)

An interactive and visually engaging **Solar System Planet Picker** built using HTML, CSS, and JavaScript.
This project uses **CSS Keyframe Animations** to create smooth transitions and dynamic planetary movements.

---

## 🚀 Features

* 🌍 Interactive planet selection
* 🎞️ Smooth CSS Keyframe animations
* 🌠 Animated transitions between planets
* 🪐 Realistic orbital motion effects
* 💡 Beginner-friendly clean code structure
* 📱 Responsive design (optional if added)
